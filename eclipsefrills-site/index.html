"Eclipse Frills" are little Eclipse plug-ins that are interesting and useful, but do not really need an entire project to themselves. They are collected here for singled download, or installation as a feature release bundled with stable versions available via the Eclipse update system.

Frill (n): Something that is desirable but not a necessity; a luxury.

The project wiki has the current and future features for all of the frills and I am always open to suggestions for new functionality and/or frills.

Features - What can I do with the frills, what benefit do they add?
Install - How can I get them and install them?
Project - Where can I find more information about the project?
Blog - What has the developer written about this project?

If you have any problems using this site or the plug-ins please feel free to submit a bug report and I will work on resolving the issues.
