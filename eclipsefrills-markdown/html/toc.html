EclipseFrills Markdown Plug-in

Overview

Markdown is a simple "markup" language that flows a little easier than HTML when writing documentation or articles. This plug-in provides a set of functionality that makes Markdown a little easier to use and interact with by providing an editor, as well as an export wizard which can export the generated HTML.

Contact
